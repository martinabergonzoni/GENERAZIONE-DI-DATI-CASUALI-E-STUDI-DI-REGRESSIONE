# GENERAZIONE-DI-DATI-CASUALI-E-STUDI-DI-REGRESSIONE
Il progetto, grazie alla generazione di età casuali, mi è servito per esercitarmi sulla creazione di campioni casuali, inferenza statistica e regressione lineare

## DESCRIZIONE DEL PROGETTO
Il progetto parte da un database generato casualmente da me ed ha l'obiettivo di farmi capire come:
- creare un campione casuale
- effettuare operazioni inferenziali
- indagare sulla correlazione tra variabili diverse
- effettuare previsioni tramite la regressione lineare semplice
